📰 NewsMag
A sleek, responsive news aggregator built with React and Vite that fetches the latest headlines from around the world in real time.

🚀 Features:
🌍 Fetches top headlines using the NewsAPI
🃏 Uniform news cards with title, description, image, and a "Read More" link
📱 Fully responsive layout using Bootstrap
⚡ Lightning-fast development with Vite

🛠️ Tech Stack:
React
Vite
Bootstrap 5
NewsApi

Installation:

1. Clone the repository

git clone https://github.com/your-username/newsmag.git
cd newsmag

2. Install dependencies

npm install

3. Start the dev server

npm run dev

🧩 Components
Navbar — a responsive navbar that shows you the category of news you can click on any category and get the news.
NewsBoard — Fetches top headlines on mount and renders them in a centered flex grid.
NewsItem — A single card showing a thumbnail, truncated title, description, and a Read More link.
